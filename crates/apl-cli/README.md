# apl

The APL command-line interface.

## Install

```bash
curl -fsSL https://apl.pub/install.sh | sh
```

## Commands

```bash
apl install <pkg>         # install packages
apl install <pkg>@<ver>   # install specific version
apl remove <pkg>          # remove packages
apl list                  # list installed packages
apl search <query>        # search for packages
apl info <pkg>            # show package details
apl update                # refresh package index
apl upgrade               # upgrade outdated packages
apl status                # check for updates
```

## Requirements

- macOS 13.0+
- ARM64 or x86_64
